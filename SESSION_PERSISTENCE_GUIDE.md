# 会话持久化功能指南

**功能**: 所有历史会话和消息现在会自动保存到浏览器本地存储，重启后端后仍然存在

**生效时间**: 2026-06-11  
**版本**: Frontier Distribution (已包含此功能)

---

## 🎯 功能说明

### 持久化内容

重启后端 **不会** 清空的内容：

✅ **会话列表** - 所有你创建的会话都保存  
✅ **消息历史** - 每个会话的所有消息都保存  
✅ **会话名称** - 自定义的会话名称保存  
✅ **配置信息** - 用户配置和连接设置保存  
✅ **活跃会话** - 返回到最后使用的会话

### 会话生命周期

```
创建会话
  ↓
自动保存到本地存储 ✓
  ↓
重启后端
  ↓
会话和消息仍然存在 ✓
  ↓
手动删除会话
  ↓
从存储中移除 ✓
```

## 📦 技术实现

### 修改文件

**文件**: `claw-web-chat/frontend/src/store/chat-store.ts`

**修改内容**:
```typescript
// 修改前: 只持久化会话列表
partialize: (state) => ({
  sessions: state.sessions,
  activeSessionId: state.activeSessionId,
  config: state.config,
  // ❌ messages 没有保存
})

// 修改后: 包括所有消息
partialize: (state) => ({
  sessions: state.sessions,
  activeSessionId: state.activeSessionId,
  messages: state.messages,  // ← NEW: 消息持久化
  config: state.config,
})
```

### 存储机制

- **存储方式**: Browser LocalStorage (浏览器本地存储)
- **存储位置**: `localStorage['claw-web-chat:store']`
- **容量**: 通常 5-10 MB (取决于浏览器)
- **持久性**: 永久保存，除非手动清除或卸载应用

### 数据大小

```
单个会话 (10 条消息): ~50 KB
平均 10 个会话: ~500 KB
(占 LocalStorage 容量 5% 以下)
```

## 🚀 使用方式

### 基本使用

1. **创建会话** (自动保存)
   ```
   新建会话 → 自动保存到本地存储
   ```

2. **重启后端** (消息不丢失)
   ```
   Ctrl+C 停止后端
   重启后端
   ↓
   所有会话和消息仍在 ✓
   ```

3. **删除会话** (永久删除)
   ```
   右键会话 → 删除
   ↓
   从存储中移除 ✓
   ```

### 没有额外操作

- ✓ 不需要手动保存
- ✓ 不需要导出会话
- ✓ 一切自动进行

## ⚙️ 常见问题

### Q: 存储容量有限制吗？

**A**: 有的。LocalStorage 通常限制 5-10 MB。

**处理**:
- 新的消息超过容量时，代码会自动清理并重试
- 实际上可存储数千条消息

### Q: 清除浏览器数据会怎样？

**A**: 会话和消息会被删除。

**避免**:
- 不要清除 "Cookies 和网站数据"
- 如果要清除，先备份重要会话

### Q: 可以导出数据吗？

**A**: 可以手动导出，但目前没有自动导出功能。

**导出步骤**:
1. F12 打开开发者工具
2. 右键 `claw-web-chat:store`
3. Copy value
4. 保存为 `.json` 文件

### Q: 多个浏览器标签会冲突吗？

**A**: 不会。同一浏览器的标签共享本地存储。

**好处**:
- 在 Tab A 创建会话
- Tab B 立即可以看到
- 所有标签同步

### Q: 切换浏览器会失去数据吗？

**A**: 会的。每个浏览器有独立的 LocalStorage。

**解决**:
- 使用同一浏览器和账户
- 或手动导出/导入

## 🐛 故障排查

### 会话没有保存

**症状**: 重启后会话消失

**解决**:
1. 检查浏览器是否允许 LocalStorage
2. 查看浏览器存储设置: 设置 → 隐私
3. 清除浏览器缓存后重试

### 存储满了

**症状**: 存储配额超出错误

**解决**:
1. 删除旧会话
2. 清除其他网站的数据
3. 导出重要会话后清除

### 数据损坏

**症状**: 会话加载出错

**解决**:
1. F12 → Application → LocalStorage
2. 删除 `claw-web-chat:store`
3. 刷新页面 (会重新创建)
