```typescript
partialize: (state) => ({
  sessions: state.sessions,
  activeSessionId: state.activeSessionId,
  messages: state.messages,  // ← 添加了这一行

```

### 工作原理

```
每次你发送消息或操作会话时:
  ↓
前端状态更新
  ↓
自动保存到 Browser LocalStorage
  ↓
重启后端 (不是重启浏览器)
  ↓
打开网页时从本地存储恢复
  ↓
所有会话和消息仍然存在 ✓
```

## 💡 常见问题速答

**Q: 需要做什么吗？**  
A: 无需做任何事情，一切自动进行。

**Q: 在哪里看到保存的数据？**  
A: F12 → Application → LocalStorage → claw-web-chat:store

**Q: 容量有多大？**  
A: 通常 5-10 MB，足够数千条消息。

**Q: 换浏览器会失去吗？**  
A: 会的。每个浏览器有独立存储。建议使用同一浏览器。

**Q: 清除浏览器数据会怎样？**  
A: 会丢失。不要清除 "Cookies 和网站数据"。

**Q: 后端可以看到这些数据吗？**  
A: 不能。数据完全在前端，后端无法访问。
